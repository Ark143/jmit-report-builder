QUICK START
================================================================================

CREATE YOUR FIRST REPORT:

1. Go to JMIT Report Builder â†’ Reports
2. Click "+ New"
3. Fill in report details:
   - Report Name: Sales Analysis
   - Data Source: SQL Query
   - Query Type: SQL
4. Enter SQL query:
   SELECT customer, SUM(amount) as total
   FROM `tabSales Invoice`
   WHERE docstatus = 1
   GROUP BY customer

5. Click "Preview Query" to test
6. Add columns in the Columns section
7. Set grouping if needed
8. Configure subtotals
9. Click "Save"
